This project got started when I couldn't find any three dimensional glassmorphism CSS tutorials while staying my portfolio website. I wanted was a glass look that had texture and a smoothly fading bevel rather than a white outline.

Achieving this effect was far fussier than I expected. So I built this generator to save you the trouble.

GitHub repo

Key characteristics

  • No white outlines!
  • Specular highlights
  • Fine-grained noise textures
  • Large blur radius
  • Bevels fade smoothly into non-existence
